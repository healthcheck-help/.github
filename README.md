This projects goal is to help you define a suitable HEALTHCHECK depending on the service you are running.

Development is happening on [Codeberg](https://codeberg.org/healthcheck) and mirrored to [GitHub](https://github.com/healthcheck-help).

## Getting started

1. Watch the introduction video:
2. Visit https://healthcheck.help for more information and resources.
3. Share your HEALTHCHECK definitions via the web app or by creating a pull request targeting the `data` repository.
4. Contribute to the project by providing feedback, suggesting improvements, or changing the code.

## About the project

At [openscript GmbH](https://openscript.ch) we often work with containerized applications. We noticed that many of our customers struggle with defining appropriate HEALTHCHECKs for their services, therefore we decided to create this project to help others in the same situation.
